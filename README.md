# Laravel-Biel

Pràctica del POL: iniciació a Laravel.

## Instalación

1. Instala dependencias PHP:
   - `composer install`
2. Crea el entorno:
   - Copia `.env.example` a `.env`
   - `php artisan key:generate`
3. Configura la base de datos en `.env` y ejecuta migraciones:
   - `php artisan migrate`
4. Frontend (si aplica):
   - `npm install`
   - `npm run dev`

## Ejecutar en local

- `php artisan serve`

## Sobre Laravel

Laravel es un framework web de PHP. Documentación oficial: `https://laravel.com/docs`.
